How to sync lyrics:
1. Select a song from the dropdown
2. Click Play to start the audio
3. When a lyric line starts, press Space (or click Tap Sync) — it stamps the current time on the next unsynced line
4. Fine-tune: click any time value to edit manually
5. Click Export to get the notes array — paste it into songs.js

Shortcuts: Space = tap sync, Z = undo last tap, P = play/pause, ← = -1s, → = +1s, Shift+← = -5s, Shift+→ = +5s
